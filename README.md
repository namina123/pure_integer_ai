# 纯整数智能体系 · PIDSLCA

**PIDSLCA**（Pure-Integer Deterministic Self-Learning Cognitive Architecture，纯整数确定性自学习认知架构）是一项探索性研究：

> **能否构建一个完全建立在整数之上、可在普通个人电脑上运行、并具备自学习能力的智能体系。**

本仓库是它的参考实现。

## 出发点

当今主流的智能系统几乎都建立在浮点运算之上——神经网络的权重是浮点，训练是浮点梯度的迭代。本项目探索另一条路径：如果把智能体系的存储与运算全部放在整数之上，会得到什么？

这样做的目的不是反对浮点，而是想看清楚一个问题：一个完全由整数构成、可在普通个人电脑上运行、能够逐位复现、并且可被审计的自学习体系，能不能成立。

## 这是什么

研究以纯整数（不依赖浮点运算）作为认知的存储与运算基底，核心包括：

- **纯整数核心**：概念、关系、强度均以整数表示，跨机器可逐位复现。
- **边计数强化（学习原语）**：关系的强度是单调不减的整数，共现计数达到阈值即被提升（tally-to-promote）。
- **六项互锁不变量**：纯整数、比特一致、单调覆写、只追加审计、教师边界、向下依赖——共同约束系统的可复现性与可审计性。
- **结构归纳**：从多个样本中通过对齐提取共享骨架，作为模板抽象。
- **构造性自检**：对教师声明的逆变换执行 B∘A 与恒等的核验，给出三值结论。
- **反剧场纪律**：每条机制都标注其真实状态（生产活 / 可选闸门 / 仅测试 / 已弃用），保证「机制接通」不等于「能力达成」。

## 当前状态

- **算术域**：端到端演示已完成——给定一份符号算术规格，系统将其编译为整数 COMPOSES 图、在整数虚拟机上以精确有理算术执行、通过重算自检结果、并通过六个可证伪的闸门。
- **语言域**：从语料中学习的机制已接通（线索涌现、对应桥、底线测量），但生成连贯句子与对话能力仍在构建中。

能力边界在论文中有完整说明。

## 路线图

正在攻克与未来计划的方向：

**近期（进行中）**

- **连贯句子生成**：目前语言域只产出词级片段，目标是端到端生成可读句子。
- **第一个统计证据**：让底线测量端到端跑通并产出 measured=True（当前卡在性能墙与若干接线缺口）。
- **性能优化**：把大规模语料训练从每轮约 15 分钟压缩到可反复实验的体量。
- **对话外壳**：在线单句输入、中文分词、渲染成句、REPL 交互——让系统能被直接对话使用。

**中期**

- **统计断奶判定**：在语言域达到「教师退场后系统仍能产出」的可证伪判据。
- **更多域与模态**：代码生成、符号数学的进一步扩展。

**长期**

- 在更多任务上检验纯整数路径的边界与适用范围。
