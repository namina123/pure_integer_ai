# 纯整数智能体系 · PIDSLCA

**PIDSLCA**（Pure-Integer Deterministic Self-Learning Cognitive Architecture，纯整数确定性自学习认知架构）是一项探索性研究：

> **能否构建一个完全建立在整数之上、可在普通个人电脑上运行、并具备自学习能力的智能体系。**

本仓库是它的参考实现。

## 这是什么

这是一份**架构存在性的探索**，不是一个成品 AI 系统。研究尝试以纯整数（不依赖浮点运算）作为认知的存储与运算基底，并以「边计数强化」（edge-count reinforcement）作为学习原语——关系的强度是单调不减的整数，共现计数达到阈值即提升。

- **纯整数核心**：概念、关系、强度均以整数表示，跨机器可逐位复现。
- **六项互锁不变量**：约束系统的可复现性与可审计性。
- **算术域**：机制的端到端演示（编译 → 整数虚拟机执行 → 自检 → 六闸门）。
- **语言域**：正在建设的第二个域，机制已接通但尚未产出可用能力。

诚实地说，它**目前还不能对话、不能生成连贯的句子**。这部分能力仍在构建中，论文对边界有完整说明。
